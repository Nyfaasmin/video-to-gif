# AI-Powered Video to GIF Generator

## How to run:
1. Install requirements:
```
pip install -r requirements.txt
```

2. Run the app:
```
python app.py
```

3. Open your browser at `http://127.0.0.1:5000/`

## What works:
- Upload MP4 file
- Enter prompt
- Generate dummy GIFs (stubbed)
